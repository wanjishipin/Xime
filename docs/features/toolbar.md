# 工具栏自定义

工具栏位于候选栏右侧，显示常用功能的快捷按钮。Xime 支持自定义工具栏按钮布局，让用户将最常用的功能放在触手可及的位置。

## 功能特性

- **自定义按钮** - 选择在工具栏中显示哪些快捷按钮
- **分页菜单** - 菜单项分页显示，横竖屏自适应布局
- **按压反馈** - 按钮按压状态背景色和透明度反馈
- **实时预览** - 自定义界面中可实时预览工具栏效果
- **主题适配** - 自动适配深色/浅色主题

## 可选的工具栏按钮

| 按钮 | 功能 |
|------|------|
| 剪贴板 | 打开剪贴板历史记录 |
| 快捷发送 | 打开快捷发送内容 |
| 表情 | 打开表情面板 |
| 管理词库 | 查看词库内容 |
| 键盘调节 | 调整键盘高度和底部间距 |
| 部署方案 | 重新加载输入方案配置 |
| 深色模式 | 切换深色/浅色主题 |
| 设置 | 打开设置页面 |

## 自定义方法

1. 点击键盘上的 Logo 按钮打开功能菜单
2. 点击菜单底部的「自定义工具栏」
3. 在自定义界面中，可以看到所有可用按钮的列表
4. 通过拖拽或开关选择要在工具栏中显示的按钮
5. 顶部预览区域会实时显示设置效果
6. 点击顶部的确认按钮保存设置

## 分页菜单

当功能菜单中的项目较多时，会自动分页显示：

- **竖屏模式** - 每页最多显示 8 个项目，底部显示圆点分页指示器
- **横屏模式** - 使用单行多列布局，充分利用横向空间
- **滑动翻页** - 在菜单页面上左右滑动可切换页面
