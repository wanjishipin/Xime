# 快捷发送

快捷发送功能允许用户收藏常用的文本内容，以便快速输入，无需重复复制粘贴。

## 功能特性

- **快速收藏** - 从剪贴板中一键收藏常用内容
- **永久保存** - 收藏的内容不会被新复制的内容覆盖
- **即时调用** - 点击即可将收藏内容插入输入框
- **轻松管理** - 支持删除不需要的收藏内容

## 使用方法

1. 进入剪贴板界面（点击 Logo 菜单 → 剪贴板）
2. 在「剪贴板」标签中找到需要收藏的内容
3. 在条目上滑动，点击星形图标添加到快捷发送
4. 切换到「快捷发送」标签，可查看所有收藏内容
5. 点击收藏内容可直接插入到输入框
6. 点击删除图标可移除收藏内容

> 从 v2.3.0-beta4 起，滑动操作面板取代了原来的长按置顶操作。在剪贴板条目上左右滑动可显示拆词、快捷发送和删除等操作选项。
