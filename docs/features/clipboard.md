# 剪贴板管理

剪贴板功能自动保存复制的内容，方便用户快速调用历史记录。

## 功能特性

- **自动保存** - 复制的文本会自动保存到剪贴板历史
- **分类查看** - 通过“剪贴板”和“快捷发送”两个标签管理
- **文本拆分（分词）** - 将剪贴板文本拆分为单个字符，逐字选择输入
- **快捷发送** - 收藏常用内容到快捷发送，永久保存
- **滑动操作** - 在项目上滑动显示拆词、快捷发送、删除等操作选项
- **批量管理** - 支持删除单条记录或清空全部历史

## 使用方法

1. 点击键盘上的 Logo 按钮打开功能菜单
2. 选择“剪贴板”进入剪贴板界面
3. 在“剪贴板”标签查看所有复制历史，在“快捷发送”标签查看收藏内容
4. 点击条目可直接插入到输入框
5. **滑动操作**：在条目上左右滑动，可显示操作面板（拆词、快捷发送、删除）
6. 点击星形图标可将内容添加到快捷发送
7. 点击删除图标可移除单条记录

## 文本拆分（分词）

文本拆分功能可将剪贴板中的一段文本按字符拆分为单个字，方便逐字选择输入：

1. 在剪贴板列表中，在需要拆分的条目上滑动
2. 点击“拆词”按钮
3. 进入分词视图，每个字符独立显示
4. 点击单个字符即可输入，点击删除可移除
5. 点击“发送”将选中的字符一次性提交
