# 键盘调整功能

Xime 输入法提供了灵活的键盘调整功能，让用户可以根据自己的使用习惯和设备特点自定义键盘的外观和行为。

## 功能特性

### 1. 键盘高度调整
Xime 支持键盘高度的动态调整，用户可以通过拖拽来改变键盘的高度：

- **拖拽调整**：在键盘界面上会出现调整控件，用户可以拖动来实时预览键盘高度
- **横竖屏独立**：横屏和竖屏可分别设置不同的键盘高度，系统会自动根据当前方向切换
- **默认高度**：键盘有默认的高度设置，可以一键恢复
- **高度范围**：键盘高度限制在屏幕高度的 1/3 到 1/2 之间，以确保可用性
- **确认保存**：调整完成后点击确认按钮，新的高度设置会被保存并应用

## 使用方法

### 进入键盘调整模式
1. 在键盘显示状态下，点击候选栏左侧的 **Xime  Logo 图标**
2. 在弹出的菜单中选择 **"键盘调节"**
3. 进入键盘调整模式后，屏幕会显示调整界面

### 键盘调整模式
键盘调整模式提供两种调整功能：

#### 1. 键盘高度调整
- 拖动键盘顶部的横条控件向上或向下
- 键盘高度会随之改变，可实时预览效果
- 高度范围限制在屏幕高度的 1/3 到 1/2 之间

#### 2. 底部留白调整
- 在键盘下方的黑色遮罩区域向上或向下拖动
- 可调整键盘与屏幕底部的间距
- 适用于全面屏手机或有底部导航栏的设备

### 操作步骤
1. 根据需要调整键盘高度或底部留白
2. 屏幕上会显示当前高度和底部留白的数值
3. 点击"重置"恢复默认设置
4. 点击"确认"保存调整结果，或"取消"放弃更改

## 横竖屏独立设置

Xime 支持横屏和竖屏方向独立设置键盘高度：

- **自动识别** - 系统会自动检测设备当前是横屏还是竖屏模式
- **独立存储** - 两个方向的键盘高度分开存储，互不干扰
- **自动切换** - 旋转设备时自动加载对应方向的键盘高度
- **适合场景** - 横屏模式下键盘可设置较小高度，为内容区留出更多空间
