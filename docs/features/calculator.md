# 计算器

Xime 的数字键盘内置计算器功能，支持数学表达式计算，计算结果可直接输入到文本框中。

## 功能特性

- **四则运算** - 支持加（+）、减（-）、乘（*）、除（/）运算
- **实时计算** - 输入表达式后自动计算并显示结果
- **结果上屏** - 点击计算结果直接插入到输入框
- **表达式中途编辑** - 支持在计算过程中修改表达式

## 使用方法

### 进入计算器模式

计算器模式集成在数字键盘中。切换到数字键盘后，键盘上方会显示计算器输入区域。

### 按键说明

| 按键 | 功能 |
|------|------|
| `+` | 加法 |
| `-` | 减法 |
| `*` | 乘法 |
| `/` | 除法 |
| `=` 或候选栏结果 | 将计算结果提交到输入框 |
| 删除键 | 删除最后一个输入的字符或运算符 |
| 数字键 0-9 | 输入数字 |

### 操作步骤

1. 切换到数字键盘
2. 使用数字键和运算符键输入表达式（如 `12+34`）
3. 候选栏会实时显示计算结果
4. 点击计算结果或按确认键将结果插入到输入框
5. 可以继续输入新的表达式进行计算

### 示例

```
输入：12 + 34
结果：46

输入：25 * 4
结果：100

输入：100 / 3
结果：33.3333
```
