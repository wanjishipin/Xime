# 表情

表情功能提供丰富的表情符号和自定义表情插件，让输入更加生动有趣。

## 功能特性

- **内置表情** - 包含笑脸、手势、动物、食物、自然、交通等多个分类的表情符号
- **表情插件** - 支持扩展表情插件，可添加颜文字、表情包、贴纸等
- **多标签页** - 顶部标签页切换，支持 Emoji 和插件分类
- **子分类标签** - 每个标签页内进一步分为子分类，横向滚动浏览
- **主题适配** - 自动检测并适配深色/浅色主题
- **分类浏览** - 按类别浏览表情，便于快速查找
- **快速插入** - 点击表情即可插入到输入框中
- **插件扩展** - 可通过安装插件获取更多表情资源

## 使用方法

1. 点击键盘上的 Logo 按钮打开功能菜单
2. 选择"表情"进入表情界面
3. **切换标签页**：在顶部标签页切换 Emoji 和插件分类
4. **浏览子分类**：在子分类标签中横向滚动浏览
5. 点击表情即可插入到当前输入框
6. 如需更多表情，可安装对应的表情插件
