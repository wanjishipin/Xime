# 部署方案

部署方案功能用于重新加载输入法配置，让修改后的设置生效。

## 使用方法

1. 点击键盘候选栏左侧的 Logo 图标打开功能菜单
2. 选择「部署方案」
3. 等待部署完成即可

## 需要部署的场景

- 切换输入方案后
- 启用或禁用方案后
- 导入新方案后
- 修改输入法设置后
- 添加或更新插件后

## 增量部署

Xime 采用智能部署机制，优化启动速度和用户体验：

- **首次使用** - 显示「正在编译词库...」提示，执行完整词库编译
- **后续启动** - 跳过提示，后台刷新注册表，速度快
- **按需触发** - 词库编译只发生在首次使用或下载新方案后，不会在每次切换输入法时触发
- **超时优化** - 移除了 30 秒部署超时限制，大词库也能完整编译

## 注意事项

- 部署时间取决于启用的方案数量和词典大小
- 如果一次性启用了过多方案（尤其是包含大词库的方案），部署时间会显著增长
- 建议只开启实际会用到的方案，不需要的方案保持关闭状态
- 部署期间键盘会自动隐藏
